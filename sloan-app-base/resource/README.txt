Keen Grammars NON-CONCATENATED
For use with Victory 1.0.0+

Note: Grammars are cached on device, if you update a grammar, 
make sure to change the key of that grammar so the device 
will recache and use the new changes.

Example: "key": "1" -> "key": "2"

Each Locale will have it's own folder of grammars,
* en_US is English grammars
* es_US is Spanish grammars
