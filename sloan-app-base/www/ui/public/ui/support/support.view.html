Numina Support Procedures

Included with Software Support Agreement (SSA) is Numina's 24/7 technical support service. For Level 1, highest severity issues, such as a down situation, please contact the Numina Group's Technical Assistance Hotline number at:

Call Numina Support: +1 (800) 695-4133

The caller will be instructed to provide contact information and brief description of the issue. Using the hotline 800 number assures you that an on-call engineer will respond to your service request and this number should be used for all urgent technical support inquiries.

Clients who have active SSA's in place are provided the benefits of:

  • 24/7 access to technical support from a Numina software support engineer
  • Priority handling for all service requests
